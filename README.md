# Ugly Christmas Sweater — Board Game Arena implementation

A [Board Game Arena](https://boardgamearena.com) adaptation of the licensed card game **Ugly Christmas Sweater** (trick-taking · card drafting · set collection · tableau building), built on the BGA Studio framework.

## Rules

- **Official rulebook (PDF):** https://boardgamegeek.com/filepage/186495/ugly-christmas-sweaters-rulebook-10
- A local copy is included at [`docs/ugly-christmas-sweater-rules.pdf`](docs/ugly-christmas-sweater-rules.pdf).
